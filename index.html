Technologies Used:

Front End Capstone

Pass A Note

A note sharing app where students can record class notes, upload images and keep track of upcoming events as well as due dates. Each user has a set of notes inside individual notebooks that correspond to a class or subject. The user can also search for friends and “follow” them. Users can view friends notes and notebooks and leave comments and ratings on notes.

Back End Capstone

Weatherly Designs

This is a brochure app for my jewelry business, Weatherly Designs. I built the app with c# and entity framework. Users can view products, tutorials and gemstones used in the products. They can also leave comments on the tutorials and on the products. Admin has the ability to add, edit and delete tutorials, products and gemstones. Admin has the ability to upload videos for tutorials and pictures for products and gemstones.
